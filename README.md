# WooCommerce Advanced Shipping Rules

Advanced shipping method for WooCommerce with flexible rules based on cart weight, cart total and product tags.

## Features

- Add a custom shipping method to WooCommerce Shipping Zones.
- Configure rules based on:
  - Cart total thresholds (e.g. free shipping over X).
  - Cart weight ranges.
  - Product tags (e.g. add surcharge for “fragile” items).
- Simple admin settings page.
- Lightweight, no external dependencies.
- Follows WordPress and WooCommerce coding standards.

## Requirements

- WordPress 6.0+
- WooCommerce 8.x+
- PHP 8.0+
